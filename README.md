# ZORBOX - Milli Sandbox Layihəsi

## Proyekt haqqında
Faylların avtomatik analizi və təhlükə dərəcəsinin müəyyənləşdirilməsi platforması.

## Texnologiyalar
- Backend: Java Spring Boot
- Frontend: React
- AI: Python (scikit-learn)
- Sandbox: Docker (Strelka, Cuckoo, CAPEv2)
- Database: PostgreSQL
- Queue: Redis

## Quraşdırma
```bash
docker-compose up -d
```
